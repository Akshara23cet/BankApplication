# Online Banking System (Java + JDBC + MySQL)

## 📌 Project Description
A console-based Online Banking System built using Java, JDBC, and MySQL.  
The project follows a layered architecture with separation of concerns.

## ✨ Features
- Create bank account
- Secure login using account number and PIN
- Deposit money
- Withdraw money
- Balance validation
- MySQL database integration

## 🧱 Project Structure
- model → Data models
- dao → Database access logic
- service → Business logic
- util → Database connection
- BankMain → Console frontend

## 🛠️ Technologies Used
- Java
- JDBC
- MySQL
- VS Code
